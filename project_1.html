Project Overview

The purpose of this web application to act as a hub for the musical artist Miximiximixing that allows users to find the artist's music between different distribution platforms. Users will be able to navigate to songs between multiple locations online using the site's links. It will also be made possible for users to find methods of communication and submit messages to contact Miximiximixing through the app.


A general overview includes the home page, about page, contact page, and music directory page. The first three pages are self-explanatory, and the fourth page will provide a list of all of the artist's music as is available through different services and provide links to the music on each service it is available on.

Page Design

Home

  • Name: Home
  • Purpose: To act as a main hub to move to other parts of the app.
  • Audience/Users: Customer
  • Content: An image gallery as well as a selection of links will be contained on the Home page.
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click links to navigate between pages
  • Special notes?: None

About

  • Name: About
  • Purpose: To give more detailed information about the artist and the work put on display through the project.
  • Audience/Users: Customer
  • Content: Most of the content of this page will be normal paragraph text.
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click the link to return to the Home page.
  • Special notes?: None

Contact

  • Name: Contact Me
  • Purpose: Provides contact information for Miximiximixing as well as access to a form to contact them directly.
  • Audience/Users: Customer
  • Content: Paragraph text and links, containing things such as email and social media accounts.
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click links to navigate between pages
  • Special notes?: None

Contact Directly

  • Name: Message Me
  • Purpose: To allow an easy to way send a direct email to Miximiximixing.
  • Audience/Users: Customer
  • Content: Form with text fields, check boxes, and buttons
  • Is data being entered?: Yes
  • Data Fields need validations?: Yes, for email to make sure the response can be found by the user if a response is sent.
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks, buttons.
  • Actions on this page: Click links to navigate between pages, send form.
  • Special notes?: This system may later change to allow the artist to view messages within the app, rather than through an email service.

Songs

  • Name: Song Directory
  • Purpose: To provide a list of every published track by the artist with links to where they can be found.
  • Audience/Users: Customer
  • Content: Contains song names and album/cover art (if it exists for the song) with links to the song on music hosting services that the particular song was uploaded to. Also contains links to show a list only containing songs from a specific year.
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click links to navigate between pages
  • Special notes?: None

Year 1/2/3/4/etc.

  • Name: Year [Insert Year Number]
  • Purpose: To provide a more concise and detailed list of songs for a specific year since the start of the artist's music production.
  • Audience/Users: Customer
  • Content: A list of music for the year in question. In addition to the inclusions of the full list, these lists will contain short musical samples for each song that can be played using an audio element. These will be omitted from the main list to avoid clutter and large amounts of loading that will only worsen as more tracks are added.
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click links to navigate between pages, play music samples
  • Special notes?: None

Platforms

  • Name: Platform Directory
  • Purpose: To provide a list of every platform that Miximiximixing can be found on.
  • Audience/Users: Customer
  • Content: Mainly hyperlinks and paragraph text
  • Is data being entered?: No
  • Data Fields need validations?: N/A
  • Buttons/Hyperlinks/Dropdowns: Hyperlinks
  • Actions on this page: Click links to navigate between pages
  • Special notes?: None

Dynamic Functionality

A photo gallery will be included. This will allow album artwork and promotional art to be viewed from the home page. In addition, forms will be used to submit messages to the artist from users and/or potential customers.
